# EDUCA SCHOOL

## installation

- composer install
- npm install

## Before migration 

- Please read  [Migration Readme.md](https://github.com/zakarianajdi/educa-school/blob/master/database/migrations/Readme.md)
- Run 'php artisan migrate' 
- Then 'php artisan db:seed' afterwards
